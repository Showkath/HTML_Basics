Explanation

  • Text Input: Used for user input, such as names or text data.
  • Password Input: Conceals the entered text, commonly used for password input.
  • Email Input: Allows users to enter email addresses, with built-in validation for email format.
  • Number Input: Accepts numeric values, with optional minimum and maximum range.
  • Radio Buttons: Allows users to select a single option from multiple choices (e.g., gender selection).
  • Checkbox: Allows users to select multiple options (e.g., interests) from a list.
  • Dropdown Select: Presents a list of options, and users can select one from the dropdown.
  • Textarea: Allows users to input multi-line text, useful for comments or longer messages.
  • Submit Button: Submits form data to a server for processing or performs a specific action.
  • Reset Button: Resets form fields to their default values.
  • Anchor Link: Creates a hyperlink to navigate to another webpage or resource.
  • Image: Displays an image on the page, with options to set width and height.
  • Shopping Cart: A simple demo of a shopping cart using checkboxes and buttons.
  • Range Input: Allows users to select a value from a range (e.g., volume control).
  • Color Input: Allows users to pick a color from the color palette.
  • Date Input: Enables users to select a date from a calendar popup.
  • Time Input: Allows users to select a time (hour and minutes) using a time picker.
  • File Input: Allows users to choose a file from their local device (e.g., an image for an avatar).
  • Search Input: Used for search input fields with built-in styles and search-specific functionalities.
  • Output Element: Used to display dynamic output based on JavaScript calculations (e.g., calculating the total price).
  • Week Input: Allows users to select a week and year from a dropdown calendar.
  • Month Input: Allows users to select a month and year from a dropdown calendar.

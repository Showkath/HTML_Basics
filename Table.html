Table Basic Examples

Example 1: Simple Table

Name Age Country
Showkath 30 India
Alex 25 Canada

This is a simple table with three columns: Name, Age, and Country.

Each row represents a person with their respective information.

Example 2: Table with Rowspan

Name Age Country
John Doe 30 USA
25

This table includes a rowspan attribute to merge cells in a row.

John Doe's name and country appear in two rows to indicate shared information.

Example 3: Table with Colspan

Name Address
John Doe 123 Main St New York

This table uses the colspan attribute to merge cells in a column.

The Address column spans two cells to include both the street address and city.
